# Installation Steps

To install the 'add_tool' MCPO Server, run the following command:

```json
{
    "MCP Server Example": {
        "command": "uvx",
        "args": [
            "--from",
            "git+https://github.com/troeske/mcp-server-example.git",
            "mcp-server"
        ]
    }
}
```
